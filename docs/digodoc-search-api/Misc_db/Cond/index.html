Module Misc_db.Cond

Module Cond that regroups functions dealing with conditions send with getElements request. Actially there are only two conditions : one that describes either an element belongs to module with specified name, another that describes either an element belongs to specific package.

type opam_cond = string

Condition that describes 'in package' property

type mdl_cond = string * string

Condition that describes 'in module' property

val make_opam_cond : 'a -> 'a

Creates condition from package name (with version)

val make_mdl_cond : 'a -> 'b -> 'a * 'b

Creates condition from module name and its package name (with version)
