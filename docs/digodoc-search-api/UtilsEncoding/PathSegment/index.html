Module UtilsEncoding.PathSegment

Module PathSegment defines encoding for path segment to call specific service.

val anti_slash : char

Anti slash character.

val sep : char

Special character used to separe different path segments. Pattern is encoded in order to remplace this character.

val sep_code : string

Name of variable for special character substituation.

val sep_var : string

Variable for special character substituation.

val decode : string -> string

decode seg decodes path segment with Uri and Ez_subst modules. Raises Failure if segment is not correctly encoded.

val encode : string -> string

encode pattern encodes pattern so it can be used as a path segement. Also encodes with Uri module.
